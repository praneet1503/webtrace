# what i used to make this 
jetbrains mono for the font 
copilot for final nails in the dot 
nextjs for the frontend
python for the backend
waybackpy for fetching data from the wayback machine
modal for backend hosting 
vercel for frontend hosting

# how to run this
clone the repo 
```zsh
git clone https://github.com/praneet1503/webtrace.git
```
then cd into the backend and frontend folders and run the respective start commands
```zsh
cd backend
pip install -r requirements.txt
pip install modal
modal serve modal_app.py
```
then your backend will be running on a modal url that you can find in the terminal and you can set that as the API_URL in the frontend .env file
```zsh
cd frontend
npm install
npm run dev
```
then your frontend will be running on http://localhost:3000 and you can open that in your browser to see the timeline. you can search for any website and see its timeline and snapshots from the wayback machine. you can also click on the snapshots to see the preview of the website at that time.
